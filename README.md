# ELIZABETH Shopify Theme

A custom Shopify theme built using Shopify Liquid.

## Features

- Responsive design
- Custom sections
- Product pages
- Collection pages
- Cart drawer
- Tailwind CSS support

## Requirements

- Shopify CLI
- Node.js (if using Tailwind CSS)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pratikshad0309-create/ELIZABETH.git
   ```

2. Open the project:
   ```bash
   cd ELIZABETH
   ```

3. Login to Shopify:
   ```bash
   shopify auth login
   ```

4. Start the development server:
   ```bash
   shopify theme dev
   ```
